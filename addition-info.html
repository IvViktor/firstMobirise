WEB-СПРАВОЧНИК

На даной странице содержатся определения терминам, а также короткие сведения о технологиях  применяемых в Web-разработке. Все статьи условно разделены на два блока: Front-End (технологии для создания оболочки, графической части сайта видимой посетителю) и Back-End (технологии для создания серверной логики работы сайта о существовании которой посетитель даже не задумываеться). Каждая статья содержит ссылки на сайт разработчика, документацию и online-tutorial. Приятного простмотра!

HTML(HyperText Markup Language) - стандартный язык разметки документов в сети Интернет. Документы состоят из HTML-страниц. HTML страницы читаются интернет-браузером, интерпретируются, после чего выводятся на экран. На сегодняшний день стандартом является HTML5, тем не менее довольно распостраненными являются HTML4.0, а также XHTML. Мы при создании сайтов используем HTML5, который является максимально оптимизированым для чтение содержимого поисковыми роботами.

  • www.w3.org/html/ - официальный сайт.
  • developer.mozilla.org/en-US/docs/Web/HTML - онлайн документация.
  • www.w3schools.com/html/ - онлайн туториал.

CSS (Cascading Style Sheets) - каскадные таблицы стилей, описания внешнего вида документа, написанного с использованием языка разметки HTML. CSS используется создателями веб-страниц для задания цветов, шрифтов, расположения отдельных блоков и других аспектов представления внешнего вида этих веб-страниц. Современный CSS3 предоставляет возможности работы с графикой, трехмерным преобразованием обьектов, форматирования изображений. При создании дизайна страниц разработчики часто прибегают к использованию существующих CSS фреймворков (BootStrap,Maxmertkit, Blueprint, Foundation).

  • www.w3.org/Style/CSS/ - официальный сайт.
  • developer.mozilla.org/en-US/docs/Web/CSS - онлайн документация.
  • www.w3schools.com/css/default.asp - онлайн туториал.
